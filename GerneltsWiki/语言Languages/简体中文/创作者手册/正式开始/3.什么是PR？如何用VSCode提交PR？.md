# 什么是PR？如何用VSCode提交PR？
### 什么是PR？
PR，即Pull Request。它允许其他人修改仓库的代码并提交给仓库的所有者\/管理员审核并合并。  
### 一个完整的PR是什么样的？
1. Fork仓库到你的Github账号。
2. Clone仓库到本地。
3. 创建分支。
4. 进行更改。
5. 推送分支。
6. 创建PR。  

听上去有点复杂？别担心！接下来我们将详细带你操作一遍！
